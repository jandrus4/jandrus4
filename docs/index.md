### Project Summary

* Team 201 is developing a modular, multipurpose aquatic drone designed for inspection and monitoring of surface and shallow water conditions, intended for applications such as ship hull inspection, coast patrols and marine wildlife monitoring. This datasheet covers subsystem B2, a rudder controller system which operated via an 8-bit PIC microcontroller. Subsystem B2 is part a large UART daisy-chain which passes messages to the rest of the team. The PCB is primarily powered by 9 volts from an external power source which gets regulated down to 3.3 volts.
* You can view the team's report [here.](https://egr314-s-2026-201.github.io/)
